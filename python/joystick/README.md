# How to deal with Xbox joystick 


1. Check available software

```bash
sudo apt-get update
sudo apt-get upgrade
```

2. Install xow driver 

[xow](https://github.com/medusalix/xow)

3. Install all required modules

```bash
pip3 install -r requirements.txt
```

4. Run python-script 

```bash
python3 joystick.py 
```
